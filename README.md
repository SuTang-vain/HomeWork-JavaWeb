# JavaWeb HomeWork

## 项目说明

本项目是 JavaWeb 课程作业，包含 JSP、Servlet、JDBC 等技术实践。

## 项目结构

- `Design/` - Tomcat + MySQL Docker 环境
- `JavaBean/` - JavaBean 示例
- `JDBC/` - JDBC 操作示例
- `jsp-workspace/` - JSP 练习项目

## 文档

- [项目技术实现详解](./JavaWeb项目技术实现详解.md)

## Docker 环境

进入 `Design` 目录启动：

```bash
cd Design
docker-compose up -d
```

- Tomcat: http://localhost:8090
- MySQL: localhost:3306
